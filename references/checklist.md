## 生成报告阶段

### 9大要素检查

- [ ] ① 书籍基本信息完整
  - [ ] 书名、作者、译者
  - [ ] 出版社、出版时间
  - [ ] ISBN、字数、定价
  - [ ] 作者简介
  - [ ] 版本信息

- [ ] ② 是否适合非凡精读馆选入
  - [ ] 赛道匹配度
  - [ ] 竞品分析（无编造信息）
  - [ ] 差异化价值（3点）
  - [ ] 主讲人适配度
  - [ ] 上线定位建议

- [ ] ③ 整本书的知识框架结构
  - [ ] 核心理论体系（每章详细展开）
  - [ ] 章节详解（表格）
  - [ ] 核心概念速查（生活化理解）

- [ ] ④ 近半年社交媒体相关热点
  - [ ] 至少3-5个热点话题
  - [ ] 每个话题：现象+关联+切入点

- [ ] ⑤ 豆瓣/微信读书用户痛点
  - [ ] 同类书籍普遍痛点（表格）
  - [ ] 目标用户痛点画像（4类用户）
  - [ ] 可能的内容风险点

- [ ] ⑥ 可补充的相关知识
  - [ ] 延伸阅读推荐
  - [ ] 当代应用延伸
  - [ ] 文化比较视角

- [ ] ⑦ 讲书大纲建议
  - [ ] 讲书策划核心原则（4大原则）
  - [ ] 分集策略（上集+下集）
  - [ ] 每章内容要点
  - [ ] 每章案例与共鸣点
  - [ ] 每章详略处理
  - [ ] 坡道设计
  - [ ] 金句埋点
  - [ ] 运营埋点

- [ ] ⑧ 综合评估与建议
  - [ ] 选书评分（6个维度）
  - [ ] 上线建议（分集、时机、标题、关键词）

- [ ] ⑨ 附录
  - [ ] 关键术语表
  - [ ] 数据来源说明
  - [ ] 推荐进阶阅读路径
